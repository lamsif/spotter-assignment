# Spotter Assignment

Assignment project for the Front-End Developer position in Spotter

## Live preview

[Flight app](https://flights-app-assignment.netlify.app)

## Commands

### Installation

`npm intall`

### Development server

`npm run dev`

### Production build

`npm run build`

## Stack used

`React`
`React Router DOM`
`Material UI`
`Emotion`
`Axios`
`Axios Cache Interceptor`
